This is a simple doom screen melting effect.
To use it, add Blit feature to your forward renderer.
Add DoomMeltingScreenEffect script to any gameobject.
Configure it.
Call DoomMeltingScreenEffect.StartScreenMelt() from somewhere.
Enjoy.
